# Note Dai 
# .pb: 通常是二进制格式的 Protocol Buffers 文件扩展名，用于高效地存储和传输结构化数据。这种格式非常紧凑，适合在带宽或存储受限的环境中使用。
# .pb.txt: 是 .pb 文件的文本格式版本。它的内容以人类可读的文本形式保存，因此便于查看和编辑。虽然文本格式比二进制格式更易于调试和手动修改，
# 但在实际应用中，它的文件大小会比二进制版本更大一些。
